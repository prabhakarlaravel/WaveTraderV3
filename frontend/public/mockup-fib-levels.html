Design B — TradingView-Style Price Lines Clean
Uses TradingView Lightweight Charts' native createPriceLine() API for Fib levels. Each level renders as a native price line with a colored badge on the price axis. Cleanest integration — no SVG overlay needed. Limited to solid/dashed lines (no shaded zones).
